# 故事神谕（Story Oracle）—— SillyTavern 扩展

一个可拖动的侧边面板：连接你自定义的 **OpenAI 兼容** 端点，针对*正在进行中*的剧情向 LLM 提问。
问答在它**独立的窗口、独立的历史**里进行——**完全不碰你的主聊天**，随时可清空。

**它能帮你做五件事**（顶部图标切换，互斥）：

- 💬 **普通聊天** —— 问剧情：「她为什么这么做」「现在好感度多少」。
- 🩺 **诊断** —— 修 MVU 状态变量（数值 / 状态栏对不上时）。
- 📖 **世界书** —— 聊世界书，或让它**直接帮你改**世界书。
- 🧭 **剧情参谋** —— 一起构思剧情走向，并可**悄悄引导**主聊天（含多拍长线「弧线」）。
- ✨ **校正** —— 一键修掉最新回复的「AI 味」（手动指定，或自动每条）。

> **v1.17.5 修复 / 改进（手机）**：剧情引导关窗后那枚**折叠的圆罗盘悬浮窗**——①静息时做成**半透明**，不再挡住下面的正文（碰一下 / 拖动时自动变回清晰）；②现在**能再次拖动**了（之前折叠成小罗盘后就拖不动，因为整枚药丸就是一颗按钮，被当成「点击」而非「拖拽」——轻点仍是展开，拖动才是移动）。
>
> **v1.17.4 新增**：自动校正 / 自动诊断跑的时候，「正在自动…」那条提示现在**点一下就能中断**（后台跑太久、或想停下时用）；中断发生在写入之前，所以这条回复不会被改动。
>
> **v1.17.3 修复**：✨ 校正时设为「保留」的标签块（比如让 AI 写在正文中段的 `<details>`），改完会**留在原来的位置**，不再被挪到回复末尾。
>
> **v1.17.2 修复**：✨ 校正应用后，主聊天那条回复现在会**立刻切到校正稿**，不用再「点一下 / 划一下」才显示（用 小白X / 翻译扩展 渲染的卡此前会停在旧回复；酒馆助手渲染的卡不受影响）。
>
> **v1.17.0 新增**：✨ 校正模式、发送失败「↻ 重试」、顶栏「⋯ 更多」菜单、世界书多选 + 条目内容预览、「仅角色相关世界书」、读取隐藏楼层。

## 安装与打开

把整个 `story-oracle/` 文件夹放进 SillyTavern 的第三方扩展目录，刷新浏览器：

- **较新版 ST**：`SillyTavern/data/<用户名>/extensions/story-oracle/`（默认 `default-user`）
- **较旧版 ST**：`SillyTavern/public/scripts/extensions/third-party/story-oracle/`

打开**魔棒菜单**（输入框旁的 🪄）→ **故事神谕**。想更快进出，可在设置里勾「在聊天输入栏显示快捷按钮」，输入框旁会多一个一键开关的 🌙。

## 设置

标题栏「⋯ 更多」菜单 → ⚙ **设置**。两种连接方式：

- **直连（Direct，默认）**：填端点 URL（如 `https://your-proxy.com/v1`，会自动补 `/chat/completions`）、API 密钥、模型。密钥明文存在 ST 的 settings.json（个人 / 自建实例无妨）。
- **连接配置文件（Profile）**：选一个你保存好的 ST 连接配置；请求走 ST 后端转发，**不会被 CORS 拦**。直连报网络 / CORS 错误时改用它。

常用选项：温度、最大 token、**上下文深度**（喂多少条最近消息，`-1` = 整段、`0` = 不喂）、是否带角色卡、系统提示词。普通模式还会**默认带实时变量状态**（stat_data），所以「好感度多少」会按真实数值回答；关掉或卡无 MVU 时它会**如实说看不到数值**而不瞎编。世界信息来源可选「全部激活的世界书」「**仅角色相关（排除全局）**」等，并可开「**读取隐藏楼层**」让它读到 `/hide` 的楼层。设置项按用途分组、可折叠，默认只展开「连接」。

## 💬 普通聊天

默认模式，问剧情即可。系统提示词每次发送时重建，所以神谕看到的永远是最新故事状态。

## 🩺 诊断模式（Diagnose）

修复 MVU（MagVarUpdate）的状态变量。神谕读取本卡的 MVU 规则、当前 `stat_data`、以及最新回复里的 `<UpdateVariable>`，对照剧情给出一份**保守的纠正补丁**；气泡下方出现 **将修复应用到状态** 按钮，通过 MVU 自己的管线应用，并可一键**撤销**。全量审计输出较长，token 上限已自动抬到 ≥4096。

**自动诊断（AUTO）⚠ 实验性。** 诊断按钮三态循环：关 → 🩺 **诊断**（手动）→ 🩺 **AUTO**（红色）。切到红色后，主聊天每来一条新回复都会在后台自动诊断并**自动应用**——有 `<UpdateVariable>` 就核验，没有就据剧情推导补上（可替代 MVU「额外模型解析」，但勿与之并用）。每轮在侧聊留一条记录（有改动的带「撤销」），并把更新写回消息、刷新状态栏。

> ⚠ 这是**唯一会自动写入 MVU 状态**的功能，故**默认关、需手动切到红色**，首次开启有提醒。拿不准就停在手动「🩺 诊断」——它只在你点了之后才出补丁。

**精选条目（可选）。** 勾「用精选条目」后可像世界书模式那样**逐条挑**要喂给诊断的条目，且**无视** ST 的启用 / 禁用状态——于是被禁用（以保持主聊天干净）的变量规则条目也能单独喂给诊断。带快捷选择与筛选，**按当前聊天记忆**。

## 📖 世界书模式（Lorebook）

**聊**世界书（里面写了什么、设定在哪条、有无矛盾、怎么扩写），或让它**直接改**世界书。进入后标题下方有一条工具栏：

- **选书**：默认「当前激活的全部世界书」，也可**勾选某一本或几本**（支持多选）。
- **条目选择器**：默认发整本，可展开后用复选框只挑几条发给神谕（带筛选 + 全选 / 仅蓝灯 / 仅绿灯 等快捷键，**👁 内容预览** 可每条显示一行摘要）。只影响发给 AI 的内容，应用时仍读整本，编辑任何条目都照常生效。

要求改动时，神谕会用中文说明改了什么，并附 `<LorebookEdit>` 区块（**分隔格式**而非 JSON，长正文放进 `<<<content … content>>>` 围栏里原样书写、无需转义）。四种操作：`create` 新建、`edit` 整条覆盖、`patch`（默认——「开头几字 || 结尾几字」锚定只换一段）、`delete`。随后出现 **将改动应用到世界书** 按钮：通过 ST 的 `saveWorldInfo` 写回、逐条列出结果、可一键**撤销**；找不到的 uid / 锚点会**安全跳过并说明原因**，绝不半途写入。正文**原样保存、不做宏替换**（`{{user}}` 等保持原样，不会把名字写死进书）。新建条目默认开「非递归」。

> 整本注入 token 可能较大，条目多时用条目选择器只挑几条即可。

## 🧭 剧情参谋模式（Story Advisor）⚠ 实验性

和神谕一起**构思剧情接下来怎么走**，并可一键让主聊天**被悄悄引导**。神谕会通读**整段**对话（无视上下文深度）找沉睡的伏笔、未兑现的线索、人物弧光的下一步，把可落地的走向列成**方案卡片**（`<StoryPlan>`：目标 + 起始迹象 + 契合点）。若先在普通模式聊出了感觉，切过来会有【🧭 把刚才的讨论整理成方案】按钮。

在卡片上选**强度**（**只铺垫** / **自然推进** / **尽快引爆**）→【开始引导】，方案就作为一条幕后指令注入主聊天（ST 的 `setExtensionPrompt`，深度可调）。之后**正常 RP 即可**——主线会逐步把剧情推向该方向，且**用户行动永远优先**（你偏离时它跟随）。采用后顶部出现**方案条**（关掉窗口会变成可拖动的「🧭 引导中」悬浮标签），可随时切强度、查看注入原文、或【完成】/【放弃】。方案存在**当前聊天**里，刷新 / 切换聊天都不丢、绝不串台。

**故事弧线（长线引导）⚠ 实验性。** 把单步方案升级成**多拍长线**：先定一条**贯穿线**（整条弧要讲的事），之后一拍接一拍走，每次过渡由一次后台调用现编下一拍注入主聊天，直到贯穿线**真正收束**才结束。新建时填表选**模式**（**透明** = 你看得到每拍指令；**盲盒** = 只看到给玩家的目标，幕后意图藏在剧透模糊后）、贯穿线、路标、篇幅、**难度**（平和 / 常规 / 凛冽，越高赌注越大，但「退出」永远在不可逆转折之前可达）、红线。任何时候都能【退出】、主聊天即刻复原。

> ⚠ 弧线比单步方案更吃模型与 token（每拍都通读整段历史 + 世界书），建议配**较强的模型**（DeepSeek / Gemini Pro / Opus）；流式、超时取消、📡 实时输出都已内置。

## ✨ 校正模式（Reply Fixer）

专门「修」**最新一条 AI 回复**：清掉 AI 八股套话、机械对白、过度精确的「数学论文腔」、把魔法写成理科、拖沓流水账，也能执行你定的角色知识边界 / 剧情护栏。**改动一律作为新的 swipe（重摇）应用**——原回复永远在「向左划一下」的地方；回复里的 `<UpdateVariable>` 区块会原样接回，**绝不弄坏 MVU 数值或状态栏**。

「校正设置」面板顶部有个下拉，切换**手动 / 自动**两套用法：

- **🖐 手动** —— 直接在输入框**用人话说要改哪里**（「这段对话写口语点」「把这句删掉」）。只动最新这条、只改你点名处，其余不动；默认带足上下文（剧情 + 卡 + 世界书 + 概要）一次性精修。改完先给一张**校正稿卡片**，满意再点 **[应用]**（这才变成 swipe）。
- **🤖 自动** —— **勾选长期想清理的目标**（5 个开关：AI 八股 / 套话、对话机械、过度精确、**魔法被写成理科**（仅奇幻，默认关）、描写拖沓 / 流水账——其余四项默认开），可再填**角色知识边界 / 剧情护栏**约束、开 **✂️ 收紧**（默认开，再精修删冗）、设**排除区**（写下 `<thinking>`、`<details>` 等标签的整段就不送去校正——「保留」的标签块改完**放回原来的位置**、「丢弃」的直接去掉）。两种触发：点 **[按目标校正最新回复]** 跑一次，或勾 **[自动校正每条新回复]** 常驻后台跑（✨ 图标染金为提示）。

> 💰 自动模式默认只发回复正文 + 目标，并带**禁词预筛 + 字数下限**——回复本就干净 / 太短时**根本不发起调用**，不会每条烧 token。

自动改完会**应用为新 swipe** 并在侧聊留记录，带 **[用原文]**（划回原稿）/ **[看改动]**（逐字对比）。配置**按当前聊天保存**，还能存成具名**套餐**（如「硬核西幻」）一键复用。

## 侧聊历史与运行概要

- **侧聊历史按聊天持久化**：每个聊天各存自己的一份，关窗 / 刷新 / 切走再回来都不丢；只有点标题栏 🗑 **清空**才清掉当前聊天的。
- **运行概要（📜，在「⋯ 更多」菜单）**：把剧情关键背景写成一段概要，存进当前聊天，之后在普通聊天与参谋模式下插在最近对话**正前方**喂给神谕——长 RP 里给它补全局背景。诊断 / 世界书模式不带。

## 对话操作

鼠标移到任意消息上，气泡角落出现小按钮：**你的消息**可 ✎ 编辑（保存并重新生成）/ 🗑 删除；**神谕的回复**可 ↻ 重新生成 / 🗑 删除。删除直接生效不再确认（这本就是独立草稿窗）。**发送失败时**（如共用 key 被 `429` 挡回），出错气泡上会显示常驻的 **↻ 重试**，点一下即可重发、不必重打（手机尤其方便）。Enter 发送、Shift+Enter 换行。

## 进阶选项

- **用补全预设作为系统提示词来源。** 除了直接写提示词，还可让神谕**沿用某个 Chat Completion 预设的结构**：首次选中会弹出**挑选清单**，逐块决定去留并可拖动排序（故事 / 世界 / 人格类默认保留，思维链 / 字数 / 文风等输出块默认去掉），保存后**冻结成快照**（之后改原预设不影响这份，点 [↺ 重新挑选] 才更新）。发送时**忠实复刻**主聊天的提示词结构（各插槽就地填入、保留每块的角色、宏实时解析且 `setvar` 会自愈不污染主聊天）。**选用预设时人格与下方系统提示词文本框都会停用。** 开了「应用剧情正则」时神谕会像主聊天一样渲染自己的回复（状态栏美化等正常显示）。
- **说话人格。** 普通 / 泉こなた / 贝阿特丽丝 / 胡蝶忍 给神谕套一层说话腔调（选「普通」= 不叠加）。**选预设时默认关**以免冲突。参谋 / 世界书模式下人格只改语气，不影响方案区块格式或写入条目的正文。诊断模式不生效。
- **自定义各模式提示词。** 设置里有个模式下拉，可查看 / 改写**诊断**与**世界书**模式的内置提示词（未改时随更新自动改进，改了就冻结，[↺ 重置为默认] 还原）。参谋仍实验性、暂不开放。
- **读取其它扩展的后台世界。** 勾「附带『世界引擎』后台世界状态」（默认开）后，神谕会在普通聊天与参谋模式读入**世界引擎 / World Engine** 维护的幕后世界状态（正在酝酿的事件、势力动向……），于是你能问「外面正在发生什么」。目前仅适配世界引擎；未识别到相关扩展时无开销。
- **机制块 / 思维链自动隐藏。** 普通 / 参谋模式会自动从显示和历史里剥掉回复内嵌的 `<UpdateVariable>` 增量块与泄漏的 `<think>` 思维链（诊断模式保留——读它们是其本职）。

## 界面

- 空对话时窗口中央有**示例问题 chip**，点一下即填入输入框（每种模式各一组）。
- 全键盘可用 + 读屏支持，尊重系统「减少动态效果」；移动端触摸目标更大。
- 诊断 / 世界书的配置栏可整条折叠（手机默认折叠、限高半屏）；破坏性操作弹 ST 主题化的确认框。
- 退出子模式会回到**进来之前**的那个模式，而不是一律掉回普通聊天。
- **移动 / 调整大小**：拖标题栏移动、拖右下角握把缩放（鼠标和触屏都支持）；位置与尺寸都记住，并自动限制在屏幕内。
